# 使用

npm i vuepress -g

npm start

所有需要展示在项目中的 md，都放在.docs文件下

├─ docs
│  ├─ README.md
│  └─ .vuepress
│     └─ config.js
│  └─ something # 这是放 md 的文件夹 相当于父路由
│     └─ product.md # 子路由
└─ package.json

如上配置后 通过 /something/product 即可访问
